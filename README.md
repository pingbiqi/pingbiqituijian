## 🧭 选型指南 (Selection Guide)

为了帮助您选择最适合的产品，请参考以下建议：

<details>
<summary><b>🔻 点击展开：如何选择无人机干扰器？</b></summary>

1.  **确认防御距离**:
    - *便携式/枪式*: 适合 500m - 2000m 的目视范围内防御，机动性强。
    - *固定式/全向*: 适合打造防御半径 1km - 5km 的永久禁飞区。
2.  **确认频段需求**:
    - 基础版通常覆盖 1.5G (GPS), 2.4G, 5.8G。
    - 如果需要防御改装无人机，建议选择全频段定制版（增加 433M, 900M 等频段）。
3.  **使用环境**:
    - 是否需要防水？是否需要长时间连续开机？我们的工业级设备支持 24/7 运行。

</details>

<details>
<summary><b>🔻 点击展开：数传电台如何达到最远距离？</b></summary>

1.  **视距 (LOS) 是关键**: 天线架设高度越高，菲涅尔区遮挡越少，距离越远。
2.  **降低空中速率**: 空中速率越低，接收灵敏度越高，距离越远（但吞吐量降低）。
3.  **高增益天线**: 配合我们要定向八木天线或高增益玻璃钢天线，效果提升显著。

</details>

## ❓ 常见问题 (FAQ)

**Q: 购买你们的干扰设备是否合法？**
A: 我们的产品主要面向公检法、监狱、考场、涉密单位及经授权的企业用户。个人用户购买请遵守当地无线电管理法律法规，严禁用于非法用途。

**Q: 屏蔽器会对基站造成干扰吗？**
A: 炳光无线的设备采用 **下行频段阻断技术**。我们严格控制发射频率，仅阻断手机接收信号，不干扰基站的上行信号，最大限度减少对公共通信设施的影响。

**Q: 你们支持 OEM/ODM 定制吗？**
A: **非常支持**。不管是更改外壳丝印、调整特定频段（如欧美频段/东南亚频段），还是开发特殊的通信协议，我们专业的研发团队都能快速响应。

**Q: 产品有保修吗？**
A: 我们提供完整的售后服务。工业级产品通常享有 1 年及以上的保修期，并提供终身技术咨询服务。
